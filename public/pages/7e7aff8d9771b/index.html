    【Java】基础学习

    之前也说要学学java，一直没动力，现在业务需要，必须得学了，冲一把

    # 类型

    # 基本类型

    • boolean 布尔型
    • byte 字节型
    • char 字符型
    • double 双精度浮点
    • float 单精度浮点
    • int 整型
    • long 长整型
    • short 短整型

    # 保留关键字

    • goto 是关键字，但不能使用
    • const 是关键字，但不能使用
    • null 空

    # 一个java类

    # 创建对象

    public class Puppy{
   public Puppy(String name){
      //这个构造器仅有一个参数：name
      System.out.println("小狗的名字是 : " + name ); 
   }
   public static void main(String[] args){
      // 下面的语句将创建一个Puppy对象
      Puppy myPuppy = new Puppy( "tommy" );
   }
}
